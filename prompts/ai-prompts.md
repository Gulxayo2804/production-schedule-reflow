# AI Prompt Documentation

AI tools were used as a collaborative assistant during development to explore algorithm design ideas and validate edge cases.

## Example Prompts

### Dependency Graph Design

Prompt:
"How can I implement topological sorting for scheduling tasks with dependencies in TypeScript?"

Purpose:
Used to review approaches for DAG scheduling and dependency ordering.

---

### Shift-aware Duration Calculation

Prompt:
"How can I calculate task duration across working shifts (pause outside working hours) in scheduling systems?"

Purpose:
Helped design the logic for pausing work outside shift hours and resuming on the next valid shift.

---

### Maintenance Window Handling

Prompt:
"How can scheduling algorithms avoid blocked time windows such as machine maintenance?"

Purpose:
Used to explore interval overlap detection and schedule adjustments.

---

## Notes

AI assistance was used primarily for:

- brainstorming algorithm approaches
- validating edge cases
- improving code structure

Final implementation decisions and code structure were written and adapted manually.
